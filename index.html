FAQ

Is this an extension?

No. It's a bookmark — a tiny piece of JavaScript you save as a bookmark. Nothing is installed on your machine.

Does it need access to my wallet?

Never. The script only reads public on-chain data and the Hyperliquid public API. No signatures, no approvals.

Which browsers work?

Any desktop browser with a bookmarks bar — Chrome, Arc, Brave, Firefox, Safari, Edge.

Is it free?

Yes, free trial while in beta. A pro tier with custom wallet lists and alerts is on the roadmap.
